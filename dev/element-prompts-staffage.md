# Element Generation Prompts — Core Staffage Figures

## How to Use

These prompts generate **background staffage figures** — simplified human forms meant to populate scenes at small scale. Unlike the detailed Core Figures pack, these prioritize:

- **Readable silhouette** over anatomical detail
- **2–3 flat colors** (not 4) — they must read clearly when small
- **Compact, clear poses** — no fine details like sandal straps or facial features
- **~40–50% canvas** — slightly smaller to encourage compact proportions

Each element has 2 variations (A & B). Run both and pick the strongest silhouette.
