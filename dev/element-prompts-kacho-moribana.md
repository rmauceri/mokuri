# Element Generation Prompts — Kacho-e & Moribana Packs

## How to Use

These prompts generate source PNGs for Mokuri's PNG→SVG pipeline (Route B in `Vector-Friendly Woodblock Illustration.md`). Read that document first — it provides the full technical grounding. These prompts are intentionally brief; they rely on that grounding.

**Every prompt uses the Short Template format:**
```
Japanese woodblock print [SUBJECT AND POSE]. Solid #00FF00 background.
4 flat zones: [hex] body, [hex] marking, [hex] accent, [hex] small details only.
ZERO outlines. Hard stencil edges. Single subject ~60% canvas. Mokuhanga. PNG 1024×768.
```

Each element has 3 variations to give compositional and pose choices. Choose one or run all three and pick the strongest result for the forge.

**Forge reminder**: After tracing, the darkest color automatically becomes the detail zone. Design accordingly — darkest zone (`detail`) must cover less than 8% of subject pixels.
