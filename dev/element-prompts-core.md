# Element Generation Prompts — Core Pack (Flora & Figures)

## How to Use

These prompts generate source PNGs for Mokuri's PNG→SVG pipeline. See `element-prompts-kacho-moribana.md` for full usage instructions and the grounding preamble in `generate_elements.py`.

Each element has 2 variations (A & B). Run both and pick the strongest result for the forge.
