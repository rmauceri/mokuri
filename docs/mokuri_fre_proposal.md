# Mokuri Studio --- First Run Experience (FRE) Proposal

## Objective

Design a first-run experience that: - Builds immediate confidence -
Delivers a meaningful creative outcome within 10 minutes - Encourages
repeat engagement - Sets up long-term retention loops

------------------------------------------------------------------------

## Core Principles

1.  **No blank canvas**
    -   Always start with something meaningful
2.  **Guide, don't teach**
    -   Use prompts and context instead of tutorials
3.  **One idea at a time**
    -   Introduce features progressively
4.  **Early success**
    -   User creates something beautiful quickly
5.  **Momentum over mastery**
    -   Encourage a second creation immediately

------------------------------------------------------------------------

## First 10-Minute Journey

### Minute 0: Entry

**Screen:** "Create your first print"

**Options:** - Misty Mountains - Ocean Waves - Spring Blossoms

**Goal:** Remove decision paralysis and spark creativity

------------------------------------------------------------------------

### Minute 1--2: First Interaction

**State:** Pre-composed scene

**Prompt:** "Drag elements to explore"

**User actions:** - Move elements - Resize or rotate

**Goal:** Immediate sense of control and success

------------------------------------------------------------------------

### Minute 2--4: Discovery

#### Step 1: Add element

Prompt: "Add something"

#### Step 2: Change palette

Prompt: "Try a different mood"

**Goal:** Deliver high-impact transformations with low effort

------------------------------------------------------------------------

### Minute 4--6: Craft

**Prompt:** "Reveal details"

**Action:** - Tap to carve - Light refinement tools

**Goal:** User feels ownership and creative authorship

------------------------------------------------------------------------

### Minute 6--7: Transition

**Prompt:** "Ready to print?"

**Goal:** Build anticipation

------------------------------------------------------------------------

### Minute 7--8: Print Moment

**Sequence:** - Screen dim - Paper overlay - Press animation - Reveal
final print

**Enhancements:** - Texture - Ink variation - Slight misregistration

**Goal:** Deliver emotional payoff

------------------------------------------------------------------------

### Minute 8--9: Ownership

**Screen:** Final print display

**Options:** - Save - Share - Continue editing

**Label:** "Your first print"

**Goal:** Create pride and attachment

------------------------------------------------------------------------

### Minute 9--10: Retention Hook

**Options:** - "Try a night version" - "Add snowfall" - "Create a rainy
scene"

**Goal:** Trigger second creation immediately

------------------------------------------------------------------------

## Session Framing

Introduce light structure:

-   Session 1: Your first print
-   Session 2: Evening scene
-   Session 3: Weather study

**Purpose:** Build habit and identity

------------------------------------------------------------------------

## Prompt System

### Types of prompts

1.  Scene-based
    -   "A quiet mountain morning"
2.  Emotion-based
    -   "A calm evening"
3.  Constraint-based
    -   "Only two colors"
4.  Transformation-based
    -   "Turn this into night"

------------------------------------------------------------------------

## Progressive Feature Exposure

  Stage   Features Introduced
  ------- ---------------------
  Start   Move elements
  Early   Add elements
  Mid     Palette switching
  Late    Carving
  End     Print

**Goal:** Avoid overwhelm while building confidence

------------------------------------------------------------------------

## UI Guidance Strategy

-   Subtle overlays instead of tutorials
-   Short prompts (3--5 words)
-   Visual cues over text
-   Always skippable

------------------------------------------------------------------------

## Retention Mechanics

### Immediate

-   Remix suggestions
-   One-tap transformations

### Short-term

-   Daily prompts
-   Weekly content drops

### Medium-term

-   Style packs
-   Seasonal themes

------------------------------------------------------------------------

## Monetization Integration

### Do NOT show paywall:

-   Before first print
-   During onboarding

### Introduce after:

-   2--3 creations
-   User explores variations

### Trigger example:

User taps locked element → contextual upgrade prompt

------------------------------------------------------------------------

## Success Metrics

-   \% users completing first print

-   # prints created in first session

-   Day 1 retention

-   Prompt engagement rate

------------------------------------------------------------------------

## Risks & Mitigations

### Risk: User confusion

**Fix:** Clear prompts and guided flow

### Risk: Drop-off after first print

**Fix:** Strong "next action" suggestions

### Risk: Feature overwhelm

**Fix:** Progressive disclosure

------------------------------------------------------------------------

## Summary

The FRE should feel like:

-   A guided creative ritual
-   A calm, rewarding experience
-   An invitation to continue

The goal is not to teach the tool--- it is to make the user feel like an
artist within minutes.
